
# 📈 Apple Stock Market Analysis
Apple Stock Market Analysis using Power BI – Developed an interactive Power BI dashboard to analyze Apple's historical stock performance. Created KPI cards, trend and volume visualizations, dividend analysis, and dynamic time-based filters. Used Power Query and DAX to transform data and generate actionable financial insights.

# 🚀 Project Overview

Apple Stock Market Analysis is an end-to-end Business Intelligence project built using Power BI to transform raw financial market data into meaningful insights.

The dashboard provides a complete view of Apple's historical stock performance by analyzing stock prices, trading volume, dividend payouts, daily returns, market sentiment, moving averages, and risk indicators.

This project demonstrates real-world skills in:

- Data Analysis
- Financial Analytics
- Power BI Dashboard Development
- DAX Calculations
- Data Modeling
- Business Intelligence Reporting

---

# 🎯 Business Objectives

The primary objectives of this project are:

- Analyze Apple's long-term stock performance.
- Monitor market trends and trading activity.
- Evaluate dividend growth and shareholder returns.
- Identify bullish and bearish market conditions.
- Track historical peaks and market drawdowns.
- Measure stock volatility and risk exposure.
- Create interactive visualizations for financial decision-making.
